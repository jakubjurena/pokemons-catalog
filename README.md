## Description

Pokemon info API

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Investigation

Few scripts for data investigation.

```bash
# Console log info about attacks from JSON
$ investigate:attacks

# Console log info about classifications from JSON
$ investigate:class

# Console log info about pokemon keys object
$ investigate:key

# Console log info about pokemon types
$ investigate:type
```

Schema is described in `pokemon-schema.drawio.svg`.
